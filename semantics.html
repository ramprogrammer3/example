WWF

The World Wide fund for Nature is an international organisation working on issues regarding the conservation, research and restoration of the enviroment, formerly named the world wildlife Fund. WWF was founded in 1961.

WWF's Panda symbol

The panda has become the symbol of WWF. the well-known panda logo of WWF originated from a panda named Chi Chi that was transferred from Beijing Zoo to the London Zoo in the same year of the establishment of WWF.

Google Chrome

Google Chrome is a web browser developed by Google, released in 2008. Chrome is the world's most popular web browser today!

Mozilla Firefox

Mozilla Firefox is an open-source web browser developed by Mozilla. Firefox has been the second most popular web browser since january, 2018.

Microsoft Edge

Microsoft Edge is a web browser developed by Microsoft, released in 2015. Microsoft Edge replaced Internet Explorer.
